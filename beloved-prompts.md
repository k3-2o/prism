## The Alzheimer's Spec Prompt

Produces a comprehensive, definitive specification document that captures every
decision, caveat, test result, architectural choice, and file detail. Use when
the project is deep enough that losing context would be catastrophic.

> Write this like an eloquent and highly intelligent senior engineer who knows
> that if he doesn't write everything down because he has Alzheimer's, he will
> forget everything the next day and lose his job. Writing this document is
> basically self-preservation tier emergency.
